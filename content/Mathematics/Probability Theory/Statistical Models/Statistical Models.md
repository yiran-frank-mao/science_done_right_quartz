**Def**  <i><u>Statistical Model (Family)</u></i>
A statistical model (family) is a collection of probability distributions on some sample space. We assume that the collection, $\mathcal{P}$, is indexed by some set $Θ$. The set $Θ$ is called the parameter space. For each $θ ∈ Θ$, let $F_\theta$ denote the corresponding member of the collection. Then a statistical model can be written as $\mathcal{P}=\{F_{\theta}\mid \theta\in\Theta\}$.

**Def**  <i><u>Parametric Model (Family)</u></i>
A statistical model (family) is parametric if $Θ \subseteq \R^k$ for some positive integer $k$. ^25685d
