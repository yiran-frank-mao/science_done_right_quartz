> [!attention]
> All the theorems and properties in this page hold for non-unital C\*-algebras as well, but we will only consider the unital case for simplicity.

## Positive Linear Functional

> [!definition] Positive Linear Functional
> Let $\newcommand{\A}{\mathcal{A}}\A$ be a (unital) [[C*-Algebras#^e4fdc6|C*-algebra]]. A linear map $\tau\colon \mathcal{A} \to \mathbb{C}$ is said to be *positive* if $\tau(a) \ge 0$ for any positive element $a \in \mathcal{A}^{+}$.

<u><b>e.g.</b></u>
1. For $\mathcal{A} = M_n(\mathbb{C})$, the trace map $\mathrm{tr}(A) = \sum_{i=1}^n a_{ii}$ is a positive linear functional.
2. For $\mathcal{A} = C[0,1]$, the map $\newcommand{\dd}{\,\mathrm{d}}\tau(f) = \int_{0}^{1} f(x) \dd x$ is positive (but it is not a [[Gelfand Representation#^6c5c63|character]]).
3. If $\mathcal{A}$ is commutative and $\tau$ is a [[Gelfand Representation#^6c5c63|character]], then $\tau$ is positive. For any $a = b^*b$, $\tau(a) = \tau(b^*b) = \tau(b^*) \tau(b) = \overline{\tau(b)}\tau(b) = |\tau(b)|^2 \ge 0$.
$\quad$

>[!proposition]
> Let $\tau\colon \mathcal{A} \to \mathbb{C}$ be a positive linear map, then it is order-preserving. That is, if $a, b \in \mathcal{A}_{\text{sa}}$ with $a \le b$, then $\tau(a) \le \tau(b)$.

> [!lemma] Properties of Positive Maps
> Let $\tau\colon \mathcal{A} \to \mathbb{C}$ be a positive linear map. Then:
> 1. $\tau(a^*) = \overline{\tau(a)}$ for all $a \in \mathcal{A}$. In particular, $\tau(h) \in \mathbb{R}$ if $h$ is self-adjoint.
> 2. (Cauchy-Schwarz Inequality) $|\tau(b^*a)|^2 \le \tau(a^*a)\tau(b^*b)$ for all $a, b \in \mathcal{A}$.
> 3. $\tau$ is a bounded linear functional, and $\|\tau\| = \tau(1_{\A})$.  
> $\quad$^8a4f1e

*Proof*
1. For a self-adjoint element $h \in \mathcal{A}_{\text{sa}}$, we have $h \pm \|h\| \cdot 1 \ge 0$. Since $\tau$ is positive, $\tau(h \pm \|h\| \cdot 1) = \tau(h) \pm \|h\|\tau(1) \ge 0$. This implies $\tau(h)$ is real. For any $a \in \mathcal{A}$, we can write $a = \text{Re}(a) + i\text{Im}(a)$. Then $\tau(a^*) = \tau(\text{Re}(a) - i\text{Im}(a)) = \tau(\text{Re}(a)) - i\tau(\text{Im}(a)) = \overline{\tau(\text{Re}(a)) + i\tau(\text{Im}(a))} = \overline{\tau(a)}$.
2. The proof for the Cauchy-Schwarz inequality is omitted here but is a standard result from the theory.
3. A special case of the Cauchy-Schwarz inequality is $|\tau(a)|^2 = |\tau(1^*a)|^2 \le \tau(a^*a)\tau(1^*1) = \tau(a^*a)\tau(1)$. Since $a^*a \le \|a^*a\| \cdot 1 = \|a\|^2 \cdot 1$, we have $\tau(a^*a) \le \|a\|^2 \tau(1)$.  Combining these gives $|\tau(a)|^2 \le \|a\|^2 \tau(1)^2$, so $|\tau(a)| \le \|a\|\tau(1)$. This shows $\tau$ is bounded and $\|\tau\| \le \tau(1)$. Since $\tau(1) = |\tau(1)| \le \|\tau\| \cdot \|1\| = \|\tau\|$, we conclude $\|\tau\| = \tau(1)$.

$\square$

> [!lemma]
> Let $\tau\colon \mathcal{A} \to \mathbb{C}$ be a linear functional such that $\tau(1_{\A})=1$. Then $\tau$ is positive if and only if $\|\tau\|=1$.

*Proof*  The forward direction follows from the [[#^8a4f1e|previous lemma]]. For the converse, assume $\|\tau\| = \tau(1) = 1$. Let $h \in \mathcal{A}$ be a positive element with $\|h\| \le 1$. We need to show $\tau(h) \ge 0$. Since $h$ is positive and $\|h\| \le 1$, the spectrum $\sigma(h) \subseteq [0, 1]$. This implies $\|1-h\| \le 1$. By our assumption, $|\tau(1-h)| \le \|\tau\| \cdot \|1-h\| \le 1$. Since $\tau(1-h) = \tau(1) - \tau(h) = 1 - \tau(h)$, we have $|1 - \tau(h)| \le 1$. This implies $\tau(h)$ lies in the closed disk of radius 1 centered at 1 in the complex plane. We also know that since $h$ is self-adjoint, $\tau(h)$ must be real. Thus, $\tau(h) \in [0, 2]$. For any positive element $a \ge 0$, we can consider $h=a/\|a\|$ (if $a \ne 0$), which gives $\tau(a/\|a\|) \ge 0$, so $\tau(a) \ge 0$.  $\square$

## States and the GNS Construction

> [!definition] State
> A *state* on a unital C\*-algebra $\mathcal{A}$ is a positive linear functional $\tau$ with norm 1. The set of all states on $\mathcal{A}$ is denoted by $S(\mathcal{A})$.
> By the previous lemmas, this is equivalent to $S(\mathcal{A}) = \{\tau \in \mathcal{A}^* \mid \tau(1) = 1 \text{ and } \|\tau\|=1 \}$.

> [!lemma]
> If $a \in \mathcal{A}$ is a self-adjoint element, then there exists a state $\tau \in S(\mathcal{A})$ such that $|\tau(a)| = \|a\|$. ^93c9c2

*Proof*  Let $\mathcal{B} = C^*(1, a)$ be the abelian C\*-algebra generated by $1$ and $a$. By the [[The Gelfand Theorem for Commutative C*-Algebras#^7594b2|Gelfand representation for abelian C*-algebras]], there exists a character $\tau_0 \in \Omega(\mathcal{B})$ such that $|\tau_0(a)| = \|a\|$. Since $\tau_0$ is a character on a unital algebra, $\tau_0(1) = 1$ and $\|\tau_0\| = 1$. By the [[The Hahn-Banach Theorem#^d7e9d9|Hahn-Banach theorem]], we can extend $\tau_0$ to a linear functional $\tau$ on all of $\mathcal{A}$ such that $\tau|_\mathcal{B} = \tau_0$ and $\|\tau\| = \|\tau_0\| = 1$. Since $\tau(1) = \tau_0(1) = 1$ and $\|\tau\| = 1$, $\tau$ is a state.  $\square$

> [!theorem] The Gelfand-Naimark-Segal (GNS) Construction
> For any state $\tau \in S(\mathcal{A})$, there exists a [[Hilbert Spaces#^ae0212|Hilbert space]] $\mathcal{H}_\tau$, a $\ast$-homomorphism (i.e., $\ast$-representation) $\pi_\tau\colon \mathcal{A} \to B(\mathcal{H}_\tau)$ such that:
> $$\tau(a^{*}a) \leq \|\pi_{\tau}(a)\|^{2}\leq \|a\|^{2},\quad \forall a \in \mathcal{A}.$$ ^f8665d

*Proof*  Let us first define a positive semi-definite sesquilinear form $(\cdot, \cdot)\colon \mathcal{A} \times \mathcal{A} \to \mathbb{C}$ by $(a, b) = \tau(b^*a)$. Let $N = \{a \in \mathcal{A} \mid (a, a) = 0\} = \{a \in \mathcal{A} \mid \tau(a^*a)=0\}$. By the [[The Gelfand-Naimark Theorem#^8a4f1e|Cauchy-Schwarz inequality]], $N$ is a closed left ideal of $\mathcal{A}$.
Consider the vector space $V = \mathcal{A}/N$. Define an inner product on $V$ by $\langle [a], [b] \rangle = (a, b)$, where $[a]$ denotes the equivalence class of $a$ in $V$. This is well-defined and gives $V$ the structure of an inner product space. Let $\mathcal{H}_\tau$ be the completion of $V$ with respect to this inner product, so that $\mathcal{H}_{\tau}$ is a Hilbert space.
For each $a \in \mathcal{A}$, define a linear map $\pi_0(a)\colon V \to V$ by $\pi_0(a)[b] = [ab]$.  This is well-defined because $N$ is a left ideal. This map is also bounded: $$\|\pi_0(a)[b]\|^2 = \|[ab]\|^2 = \tau((ab)^*ab) = \tau(b^*a^*ab).$$Since $a^*a \le \|a^*a\| \cdot 1 = \|a\|^2 \cdot 1$, we have $b^*a^*ab \le \|a\|^2 b^*b$. By positivity of $\tau$, $\tau(b^*a^*ab) \le \|a\|^2 \tau(b^*b) = \|a\|^2 \|[b]\|^2$. Thus $\|\pi_0(a)\| \le \|a\|$. Since $\pi_0(a)$ is a bounded linear operator on a dense subspace of $\mathcal{H}_\tau$, it extends uniquely to a bounded operator $\pi_\tau(a) \in B(\mathcal{H}_\tau)$ with $\|\pi_\tau(a)\| \le \|a\|$. The map $\pi_\tau\colon \mathcal{A} \to B(\mathcal{H}_\tau)$ is a $\ast$-homomorphism. $\square$

## The Gelfand-Naimark Theorem

> [!definition] Direct Sum of Hilbert Spaces
> Let $\{\mathcal{H}_i\}_{i \in I}$ be a collection of [[Hilbert Spaces#^ae0212|Hilbert spaces]]. The *direct sum* $\mathcal{H} = \bigoplus_{i \in I} \mathcal{H}_i$ is the Hilbert space of elements $(\xi_i)_{i \in I}$ where $\xi_i \in \mathcal{H}_i$ and $\sum_{i \in I} \|\xi_i\|^2 < \infty$.
> If for each $i \in I$, we have an operator $a_i \in B(\mathcal{H}_i)$ such that $\sup_i \|a_i\| < \infty$, we can define the *direct sum of operators* $a = \bigoplus_{i \in I} a_i \in B(\mathcal{H})$ by $a(\xi_i) = (a_i \xi_i)$. Then $\|a\| = \sup_i \|a_i\|$.

> [!theorem] Gelfand-Naimark Theorem
> Any unital C\*-algebra $\mathcal{A}$ is isometrically $\ast$-isomorphic to a subalgebra of $B(\mathcal{H})$ for some [[Hilbert Spaces#^ae0212|Hilbert space]] $\mathcal{H}$.

*Proof*  For each state $\tau \in S(\mathcal{A})$, perform the [[The Gelfand-Naimark Theorem#^f8665d|GNS construction]] to get a representation $(\pi_\tau, \mathcal{H}_\tau)$. Define the Hilbert space $\mathcal{H} = \bigoplus_{\tau \in S(\mathcal{A})} \mathcal{H}_\tau$, and the *universal representation* $\pi\colon \mathcal{A} \to B(\mathcal{H})$ by $\pi(a) = \bigoplus_{\tau \in S(\mathcal{A})} \pi_\tau(a)$.  This is a well-defined $\ast$-homomorphism because $\|\pi(a)\| = \sup_{\tau \in S(\mathcal{A})} \|\pi_\tau(a)\| \le \sup_{\tau \in S(\mathcal{A})} \|a\| = \|a\|$.
We now show that $\pi$ is an isometry. For any $a\in\mathcal{A}$, since $\|\pi(a)\| =\sup_{\tau \in S(\mathcal{A})} \|\pi_\tau(a)\|$, from the GNS construction, for any state $\tau$,  $\sup_{\tau\in S(A)} \tau(a^{*} a) \leq \| \pi(a) \|^{2} \leq \|a\|^2$ for any $a$. On the other hand, by [[The Gelfand-Naimark Theorem#^93c9c2|the lemma above]], for the self-adjoint element $a^*a$, there exists a state $\tau_0 \in S(\mathcal{A})$ such that $\tau_0(a^*a) = \|a^*a\| = \|a\|^2$. So, $\|\pi(a)\|^2 \ge \tau_0(a^*a) = \|a\|^2$, and thus $\|a\|^{2} = \|\pi(a)\|^2$, $\pi$ is an isometry. An isometric $\ast$-homomorphism is [[Relations and Functions#^042daf|injective]], so $\mathcal{A}$ is isometrically $\ast$-isomorphic to its image $\pi(\mathcal{A})$. $\square$
