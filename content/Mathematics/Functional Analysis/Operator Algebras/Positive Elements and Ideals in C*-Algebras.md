## Positive Elements

> [!definition] Positive Element
> An element $a$ in a unital C\*-algebra $\newcommand{\A}{\mathcal{A}}\A$ is said to be *positive* if it is self-adjoint and its spectrum $\sigma(a)$ consists of non-negative real numbers, i.e., $\sigma(a) \subseteq [0, \infty)$. The set of all positive elements in $\A$ is denoted by $\A^{+}$.

<u><b>e.g.</b></u>  Let $X$ be a locally compact [[Separation and Hausdorff Spaces#^f7bcc8|Hausdorff]] space. Then $f\in C_{0}(X)$ is positive if and only if either $f(x)\geq 0$ for all $x\in X$ (i.e. $\sigma(f)\subset \mathbb{R}_{+}$) or $f=\overline{g}g$ for some $g\in C_{0}(X)$.

Our main goal in this section is to show that the set of positive elements is precisely the set of elements of the form $x^*x$ for some $x \in \A$.

> [!proposition]
> For any element $a$ in a C\*-algebra $\A$, $a$ is positive if and only if $a=b^{*}b$ for some $b\in\mathcal{A}$.

*Proof* We prove this in two steps.
1. **Step 1:** We show that if $-x^*x$ is a positive element, then $x$ must be zero. 
   If $-x^*x \in A^+$, its spectrum $\sigma(-x^*x)$ is contained in $[0, \infty)$. We also know that for any $x$, the spectrum of $x^*x$ is contained in $[0, \infty)$. Since $\sigma(-x^*x) = \{-\lambda \mid \lambda \in \sigma(x^*x)\}$, the only way for both conditions to hold is if $\sigma(x^*x) = \{0\}$. Since $x^*x$ is self-adjoint, its norm is equal to its spectral radius. Therefore, $\|x^*x\| = r(x^*x) = 0$, which implies $x^*x = 0$. By the C*-identity, $\|x\|^2 = \|x^*x\| = 0$, so $x=0$.
2. **Step 2:** Let $b = x^*x$. As $b$ is self-adjoint, we can decompose it into its positive and negative parts: $b = b^+ - b^-$, where $b^+, b^- \in A^+$ and $b^+b^- = 0$. Our goal is to show that $b^-=0$. 
   Consider the element $c = xb^-$. We compute $-c^*c$: $$-c^*c = -(xb^-)^*(xb^-) = -b^-x^*xb^- = -b^-bb^-$$Substituting $b = b^+ - b^-$, we get: $$-c^*c = -b^-(b^+-b^-)b^- = -b^-b^+b^- + (b^-)^3$$
   Since $b^+b^-=0$, we have $b^-b^+ = (b^+b^-)^* = 0$, so the first term vanishes. Thus, $-c^*c = (b^-)^3$. Since $b^-$ is positive, $(b^-)^3$ is also positive. We now have that $-c^*c$ is a positive element. By Step 1, this implies $c=0$. If $c = xb^- = 0$, then $c^*c = (xb^-)^*(xb^-) = b^-x^*xb^- = b^-bb^- = (b^-)^3 = 0$. Since $b^-$ is positive, $(b^-)^3=0$ implies $b^-=0$. 
   Therefore, $b = b^+$, which means $x^*x$ is a positive element. 

$\square$

> [!theorem] Unique Positive Square Root
> An element $a \in \A$ is positive if and only if there exists a unique positive element $b \in \A^+$ such that $a = b^2$. This element $b$ is called the *positive square root* of $a$, denoted by $a^{1/2}$.

*Proof*
* **Existence:** If $a \in \A^{+}$, then $\sigma(a) \subseteq [0, \infty)$. The square root function $f(t) = \sqrt{t}$ is continuous on $\sigma(a)$. By the [[The Gelfand Theorem for Commutative C*-Algebras#^b9e7cb|continuous functional calculus]], we can define $b = f(a) = a^{1/2}$.
    * Since $f$ is a real-valued function, $b$ is self-adjoint.
    * The spectrum of $b$ is $\sigma(b) = f(\sigma(a)) = \{\sqrt{\lambda} \mid \lambda \in \sigma(a)\} \subseteq [0, \infty)$, so $b \in A^+$.
    * By the properties of functional calculus, $b^2 = (f \cdot f)(a)$. Since $(f \cdot f)(t) = (\sqrt{t})^2 = t$, we have $b^2 = a$.
* **Uniqueness:** Suppose there is another element $c \in A^+$ such that $c^2 = a$. Since $c$ is a polynomial in itself ($c^2=a$), it commutes with $a$. The element $b=a^{1/2}$ is a limit of polynomials in $a$, so $c$ also commutes with $b$. Now consider the commutative C\*-algebra $B = C^*(b,c)$ generated by $b$ and $c$. By the Gelfand representation theorem, there is a $*$-isomorphism $\hat{\cdot} : B \to C(\Omega)$ for some compact Hausdorff space $\Omega$. We have $\hat{b}^2 = \hat{a}$ and $\hat{c}^2 = \hat{a}$. Since $b,c \in A^+$, their Gelfand transforms $\hat{b}$ and $\hat{c}$ are non-negative real-valued functions. From $\hat{b}^2 = \hat{c}^2$, we can conclude that $\hat{b} = \hat{c}$. Since the Gelfand map is an isomorphism, this implies $b=c$. 

$\square$

> [!definition] Absolute Value and Positive/Negative Parts
> For any self-adjoint element $a \in \A$, we define:
> * The **absolute value** of $a$ as $|a| := (a^2)^{1/2}$.
> * The **positive part** of $a$ as $a^+ := \frac{1}{2}(a + |a|)$.
> * The **negative part** of $a$ as $a^- := \frac{1}{2}(|a| - a)$.
>
> These elements are positive and satisfy $a = a^+ - a^-$ and $a^+a^- = 0$.

## The Partial Order on Self-Adjoint Elements

> [!lemma]
> If $a, b \in \A^+$, then $a+b \in \A^+$.

*Proof*  Since $a$ and $b$ are positive, they are self-adjoint, so $a+b$ is self-adjoint. The proof that $\sigma(a+b) \subseteq [0, \infty)$ uses the fact that for $x \in A_{sa}$, $x \in A^+$ if and only if $\|\lambda \cdot 1 - x\| \le \lambda$ for all $\lambda \ge \|x\|$. Let $s = \|a\|$ and $t = \|b\|$. Then we have $\|s \cdot 1 - a\| \le s$ and $\|t \cdot 1 - b\| \le t$.
By the triangle inequality: $$\|(s+t) \cdot 1 - (a+b)\| = \|(s \cdot 1 - a) + (t \cdot 1 - b)\| \le \|s \cdot 1 - a\| + \|t \cdot 1 - b\| \le s+t$$Since $s+t \ge \|a+b\|$, this shows that $a+b$ is positive. $\square$

> [!definition] The Partial Order on $\A_{\text{sa}}$
> Let $\A_{\text{sa}}$ be the set of self-adjoint elements in $\A$. For $a, b \in \A_{\text{sa}}$, we write $a \le b$ if $b-a \in \A^{+}$. This defines a [[Order#^fa7f22|partial order]] on $\A_{\text{sa}}$.

## Ideals and Quotient Algebras

> [!theorem] Ideals and Quotients of C*-Algebras
> Let $I$ be a closed two-sided ideal in a C*-algebra $\A$. Then:
> 1. $I$ is a self-adjoint ideal (i.e., a $\ast$-ideal).
> 2. The quotient algebra $\A/I$ is a C*-algebra with the quotient norm.
> $\quad$

To prove this, we first need a result about approximate identities.

> [!lemma] Approximate Identity
> For any $a \in A$ and $\epsilon > 0$, there exists an element $e \in I \cap A^+$ with $\|e\| \le 1$ such that $\|a - ae\| < \epsilon$ and $\|a - ea\| < \epsilon$. Such a family of elements is called an *approximate identity* for $A$. If $I$ is an ideal, it contains an approximate identity for itself.

*Proof of the Theorem*
1. **$I$ is a $\ast$-ideal:** We need to show that if $a \in I$, then $a^* \in I$.
   Since $a \in I$ and $I$ is a right ideal, $a^*a \in I$. Because $I$ is an ideal, it contains an approximate identity $\{e_n\}$. These $e_n$ can be constructed via functional calculus on elements like $a^*a$, and since $I$ is closed, these elements $e_n$ lie in $I$. 
   We know that $\lim_{n \to \infty} \|a - ae_n\| = 0$. Then, using the property that the involution is an isometry: $$\|a^* - e_n a^*\| = \|(a - ae_n)^*\| = \|a - ae_n\|$$So, $\lim_{n \to \infty} \|a^* - e_n a^*\| = 0$. Each element $e_n a^*$ is in $I$ because $e_n \in I$ and $a^* \in A$. Since $a^*$ is the limit of a sequence of elements in $I$ and $I$ is a closed subspace, we must have $a^* \in I$. Thus, $I$ is a $\ast$-ideal.
2. **$A/I$ is a C\*-algebra:** Since $I$ is a closed ideal in a Banach algebra $A$, the quotient $A/I$ is a Banach algebra with the quotient norm $\|x+I\| = \inf_{b \in I} \|x+b\|$. Since $I$ is a $\ast$-ideal, $A/I$ is a Banach $\ast$-algebra. We only need to verify the C\*-identity: $$\|a^*a + I\| = \|a+I\|^2 \quad \text{for all } a \in A$$First, for any $b \in I$: $$\|a^*a+I\| = \inf_{c \in I} \|a^*a+c\| \le \|(a+b)^*(a+b)\| = \|a+b\|^2$$Taking the infimum over all $b \in I$ gives $\|a^*a+I\| \le \|a+I\|^2$.
   For the reverse inequality, let $\{e_n\}$ be an approximate identity for $I$. Then $\|a+I\| = \lim_{n \to \infty} \|a(1-e_n)\|$. We compute the square: $$\|a(1-e_n)\|^2 = \|(a(1-e_n))^* a(1-e_n)\| = \|(1-e_n)a^*a(1-e_n)\|$$Since $e_n \in I$, the element $c_n = e_n a^*a (1-e_n) + a^*a e_n$ is in $I$. Thus, $(1-e_n)a^*a(1-e_n) = a^*a - c_n$.Therefore, for each $n$: $$\| (1-e_n)a^*a(1-e_n) \| = \|a^*a - c_n\| \ge \inf_{c \in I} \|a^*a-c\| = \|a^*a+I\|$$So, $\|a(1-e_n)\|^2 \ge \|a^*a+I\|$ for all $n$. Taking the limit as $n \to \infty$, we get: $$\|a+I\|^2 = \lim_{n \to \infty} \|a(1-e_n)\|^2 \ge \|a^*a+I\|$$Combining the two inequalities, we obtain the C\*-identity. 

$\square$
