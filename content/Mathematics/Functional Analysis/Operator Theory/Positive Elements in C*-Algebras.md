> [!definition] Positive Element
> $a\in\mathcal{A}$ is positive if $a=b^{*}b$ for some $b\in\mathcal{A}$.

> [!proposition]
> If $\mathcal{A}$ is a C\*-algebra, then $a\in\mathcal{A}$ is positive if and only if $a$ is s.a. and $\sigma(a)\subseteq \R_{+}=[0,\infty)$.
> 


<u><b>e.g.</b></u>  Let $X$ be a locally compact [[Hausdorff Space#^f7bcc8|Hausdorff]] space. Then $f\in C_{0}(X)$ is positive if and only if either $f(x)\geq 0$ for all $x\in X$ (i.e. $\sigma(f)\subset \R_{+}$) or $f=\overline{g}g$ for some $g\in C_{0}(X)$.
