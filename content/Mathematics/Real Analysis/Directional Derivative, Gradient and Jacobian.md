## Extreme Values

> [!theorem] Second Derivatives Test
> If $f\colon \R^{2} \to\R$ has continuous second order partial derivatives in a neighbourhood of $(a,b)$ and $\newcommand{\pddf}[2]{\frac{\partial#1}{\partial#2}} \pddf{f}{x}(a,b)=\pddf{f}{y}(a,b)=0$. Let $D=\pddf{^2 f}{x^2}(a,b)\pddf{^2 f}{y^2}(a,b)-\left(\pddf{^2 f}{x\partial y}(a,b)\right)^2$.
> - If $D>0$ and either $\pddf{^2 f}{x^2}(a,b)>0$ or $\pddf{^2 f}{y^2}(a,b)>0$, then $f$ has a local minimum at $(a,b)$.
> - If $D>0$ and either $\pddf{^2 f}{x^2}(a,b)<0$ or $\pddf{^2 f}{y^2}(a,b)<0$, then $f$ has a local maximum at $(a,b)$.
> - If $D<0$, then $f$ has a saddle point at $(a,b)$.
> - If $D=0$, the test is inconclusive. i.e., we need to consider higher order derivatives to determine.
> $\quad$
